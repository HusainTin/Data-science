# Data Science Projects

This repository contains a collection of data science projects and analyses, primarily focused on machine learning, computer vision, and data analysis using Python and Jupyter notebooks.

## Projects

### 1. Heap Implementation
- **File**: `heap.ipynb`
- **Description**: Implementation and analysis of heap data structures.

### 2. MNIST Analysis
- **File**: `mnist_analysis.ipynb`
- **Description**: Analysis and classification of the MNIST handwritten digit dataset.

### 3. CIFAR Image Analysis
- **Directory**: `cifar-image-analysis/`
- **Files**: `analysis.ipynb`
- **Description**: Analysis of the CIFAR-10/CIFAR-100 image classification datasets.

### 4. Customer Churn Analysis
- **Directory**: `customer churn/`
- **Files**: `analysis.ipynb`, `customer_churn.csv`
- **Description**: Predictive modeling for customer churn using the provided dataset.

### 5. Data Augmentation
- **Directory**: `data-augmentation/`
- **Files**: `analysis.ipynb`, `transfer_learning.ipynb`
- **Datasets**: Flower photos dataset
- **Description**: Techniques for data augmentation and transfer learning applications.

### 6. House Price Analysis
- **Directory**: `house-price-analysis/`
- **Files**: `house-price-analysis.ipynb`, `House_Price.csv`
- **Description**: Analysis and prediction of house prices using regression models.

## Setup

1. Ensure you have Python 3.7+ installed.
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```

4. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## Usage

Each project contains a Jupyter notebook with detailed analysis, code, and explanations. Open the respective `.ipynb` files in Jupyter to explore the analyses.

## Logs

Training logs and model checkpoints are stored in the `logs/` directory.
